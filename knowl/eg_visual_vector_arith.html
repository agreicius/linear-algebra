Example 3.1.7. Visualizing vector arithmetic in \(\R^2\) and \(\R^3\).

Fix \(n\in\{2,3\}\text{.}\) Once we choose a coordinate system for \(\R^n\) (complete with origin and coordinate axes), the standard way of visualizing a vector \(\boldv=(a_1,a_2,\dots, a_n)\) is to sketch it as the position vector \(\overrightarrow{OP}\text{,}\) represented as an arrow from the origin \(O=(0,0,\dots, 0)\) to the point \(P\) with coordinates \((a_1,a_2,\dots, a_n)\text{.}\) With this convention in place, vector addition can be visualized using the familiar “tip to tail” method; and vector scalar multiplication can be understood as scaling the position vector in the proper direction.
