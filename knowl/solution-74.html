Solution 3.7.8.1.

First we use ProcedureÂ 3.6.13 to produce a basis of \(W\text{.}\) A general element of \(W\) has parametric description
\begin{equation} p(x)=ax^3+bx^2+cx+a=a(x^3+1)+bx^2+cx\text{,}\tag{3.7.2} \end{equation}
from which we see immediately that \(B=\{x^3+1, x^2, x\}\) is a spanning set of \(W\text{.}\) From (3.7.2) we also see that
\begin{align*} a(x^3+1)+bx^2+cx=\boldzero \amp \iff ax^3+bx^2+cx+a=0x^3+0x^2+0x+0 \\ \amp \iff a=b=c=0\text{.} \end{align*}
Thus \(B\) is a basis of \(W\text{.}\) Since \(B\) contains three distinct elements, we conclude that \(\dim W=\abs{B}=3\text{.}\)
