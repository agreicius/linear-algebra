Theorem 3.2.9. Matrix transformations I.

Let \(A\) be an \(m\times n\) matrix. The function \(T_A\colon \R^n\rightarrow \R^m\) is a linear transformation.

Proof.

We use the one-step technique. For any \(c,d\in \R\) and \(\boldx_1, \boldx_2\in \R^n\text{,}\) we have
\begin{align*} T_A(c\boldx_1+d\boldx_2) \amp =A(c\boldx_1+d\boldx_2)\\ \amp =A(c\boldx_1)+A(d\boldx_2) \amp (\knowl{./knowl/xref/th_matrix_alg_props.html}{\text{Theorem 2.2.1}}) \\ \amp =cA\boldx_1+dA\boldx_2 \amp (\knowl{./knowl/xref/th_matrix_alg_props.html}{\text{Theorem 2.2.1}})\\ \amp =cT_A(\boldx_1)+dT_A(\boldx_2)\text{.} \end{align*}
This proves \(T_A\) is a linear transformation.
