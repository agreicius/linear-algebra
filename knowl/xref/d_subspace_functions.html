Definition 3.3.20. Function subspaces.

Let \(I\subseteq \R\) be a nondegenerate interval.
  1. We denote by \(C(I)\) the set of all continuous functions on \(I\text{:}\) i.e.,
    \begin{equation*} C(I)=\{f\in F(I,\R)\colon f \text{ is continuous on } I\}\text{.} \end{equation*}
